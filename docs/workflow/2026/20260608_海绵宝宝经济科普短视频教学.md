## 1、使用豆包生成文案

​	　首先，我们需要找到此类视频的一个爆款视频，然后，使用豆包提取它的文案，并根据爆款文案的结构重新生成新的文案。

::: warning 为什么要提取爆火文案

因为爆火文案的文案结构、写作手法等都是有出彩之处的，借助爆款文案内容框架，更容易写出优秀的文案

:::

​	　以抖音爆款视频：[什么是五险一金](https://www.douyin.com/user/MS4wLjABAAAAqfJRiJ421qnrLdzdOWo1YeOZzFWwKjwdIEepW7PIMpKrIQgLS3IIE5dUm_B6LEi4?from_tab_name=main&modal_id=7616638613559844985) 为例，首先，在视频处点击`分享` -> `复制分享链接`，再将复制的链接发送给豆包，让其提取文案。

::: code-group

```text{2} [提示词]
0.53 10/18 xsR:/ :7pm a@n.QX 什么是五险一金？# 科普 # 动画 # 财经 # 金融 # 海绵宝宝https://v.douyin.com/XLHxrfRq3TM/ 复制此链接，打开Dou音搜索，直接观看视频！
提取文案
```

``` text{9-16} [提示词]
什么是五险一金？
海绵宝宝在蟹堡王天天勤勤恳恳做蟹黄堡，蟹老板却抠门不给交五险一金，海绵宝宝还傻傻不知道自己亏大了。其实五险一金就是打工人的五大生活保障加一笔买房巨款。

学习该文案的写法手法和待点，采用海绵宝宝的世界观，以海绵宝宝为主角，解释一下有限公司的概念
故事必须全新原创，和参考文案的走剧情走向完全无关，重新设计一个全新的商业小故事
必须加入参考文案没有的知识点
语言风格只借鉴短视频口语化节奏，句式、用词、比喻全部重新创作
开头钩子、中间发展、结尾总结，全部重新设计，不使用原文任何结构

## 7、使用豆包生成图片

​	　接下来，就是根据文案生成相匹配的人物或者建筑等进行展示，在加上一些动画即可。此外，还可以使用文本、贴纸、音效等丰富视频元素。

（1）豆包生图提示词

```
方法1：使用豆包生成单个角色不同表情
以海绵宝宝世界观，生成透明背景的图片 
1、只有海绵宝宝一个人，没有其他背景 
2、生成海绵宝宝开心、难过、惊讶的表情的图片
3、生成海绵宝宝蹲在灶台边，满脸好奇的图片

方法2：直接根据文案生成
以海绵宝宝世界观，根据以下文案，生成透明背景的图片： 
描述：供货商堵在门口讨要大额欠款

方法3：在互联网搜索类似图片，或者直接在动画中截图
然后在豆包使用下面提示词：
帮我生成图片，把主题完整的扣除，生成9：16的透明底图
